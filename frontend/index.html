Platform Connections

How to use TalkSync with each platform
📹
Google Meet
Full real-time overlay via Chrome Extension
✅ Fully Supported

1. Install the TalkSync Chrome Extension
2. Join a Meet call
3. Click the 🎙 icon in toolbar
4. Paste Gemini key → Start
5. Overlay appears on Meet screen

🎥
Zoom
Use side-by-side with TalkSync web app
🔵 Side-by-Side Mode

1. Open this app in Chrome
2. Go to Live Call tab
3. Open Zoom in separate window
4. Press Start — speak in Zoom,
   read translation here in real-time

💼
Microsoft Teams
Use side-by-side with TalkSync web app
🔵 Side-by-Side Mode

1. Open this app in Chrome
2. Go to Live Call tab
3. Open Teams in separate window
4. Press Start — mic captures
   everything said in the meeting

🌐
Any Browser Call
Works with any web-based video call
✅ Universal

This app uses your system microphone.
It picks up all audio in your environment — so it works with any meeting tool simultaneously.
